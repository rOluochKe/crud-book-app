# React Query 3 - CRUD Book App

You need to launch client and server separately.

To launch client go to `client` folder and run:

```
yarn && yarn start
```

Then in a separate tab open `server` and run:

```
yarn && yarn start
```

## Technoogies

- React
- React Query
- Node
- Express
- Styled Component
- Swagger
- Rebase
